## Strategy

For slides with >600px overflow after initial fixes:
- Consider if content can be split into multiple slides
- Use ultracompact as last resort
- Radically simplify while keeping core message
- Remove non-essential examples

For 400-600px range:
- More aggressive condensing with supercompact
- Combine related items
- Use shorter synonyms

## Notes

The key is finding the right balance between:
- Eliminating overflow
- Maintaining readability (prefer 14px supercompact over 12px ultracompact)
- Preserving educational value
