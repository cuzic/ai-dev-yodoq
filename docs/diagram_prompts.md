# セミナースライド用図表作成プロンプト集

## カラースキーム

全ての図表は以下のブランドカラーを使用してください：

**ベースカラー:**
- **トリトンブルー**: `#00146E` (RGB: 0, 20, 110) - 濃い青。ブランドメインカラー
- **モヒカンブルー**: `#00AFF0` (RGB: 0, 175, 240) - 薄い青。機体ラインやサブ色
- **ホワイト**: `#FFFFFF` (RGB: 255, 255, 255) - ブランドの余白・背景色

**派生カラー（必要に応じて使用）:**
- **トリトンブルー（薄）**: `#003399` - トリトンブルーの中間色
- **トリトンブルー（極薄）**: `#4D6DB3` - トリトンブルーの明るい派生色
- **モヒカンブルー（薄）**: `#66D9FF` - モヒカンブルーの明るい派生色
- **アクセントグレー**: `#E6E6E6` - 背景やボーダー用
- **テキストグレー**: `#333333` - 本文テキスト用

**使用ガイドライン:**
- メイン要素: トリトンブルー
- サブ要素・強調: モヒカンブルー
- 背景: ホワイト
- 補助的な要素: 派生カラー

---

## 使い方
各プロンプトをClaude Codeに渡して、SVG形式で図を生成してください。
生成されたSVGを該当スライドに挿入します。
